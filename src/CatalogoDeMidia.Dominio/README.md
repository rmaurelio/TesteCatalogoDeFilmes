# CatalogoDeMidia.Dominio

Este projeto contém as entidades de domínio, enums e interfaces de repositório para o sistema de catálogo de mídias.

## Estrutura de Pastas

- **Entidades**: Contém as entidades de domínio (ex.: `Midia`)
- **Enums**: Contém os tipos enumerados (ex.: `TipoMidia`)
- **Repositorios**: Contém as interfaces de repositório (ex.: `IMidiaRepositorio`)

## Responsabilidades

- Definir as entidades e regras de negócio essenciais
- Definir enums e tipos de valor
- Definir interfaces de repositório sem conhecimento de implementação

## Dependências

Este projeto não depende de nenhum outro projeto da solução.
